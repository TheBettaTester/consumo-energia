## 📌 Sobre o projeto

Este projeto é uma calculadora de consumo de energia elétrica desenvolvida em Python.

O objetivo é permitir que o usuário calcule o consumo de aparelhos e estime o custo da energia com base no uso diário.

---

## 🛠️ Tecnologias utilizadas

- Python
- VS Code
- GitHub

---

## ⚙️ Funcionalidades

- Cálculo de consumo diário, semanal, mensal e anual
- Estimativa de custo em reais (R$)
- Classificação do consumo:
  - Baixo consumo
  - Médio consumo
  - Alto consumo
- Validação de entradas do usuário

---

## 📐 Fórmula utilizada

Consumo (kWh) = (Potência (W) × Horas por dia) / 1000

Custo = Consumo × valor do kWh
