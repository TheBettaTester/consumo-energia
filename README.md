## 💡 Sobre o projeto

Este projeto é uma calculadora de consumo de energia elétrica desenvolvida em Python.

O objetivo é permitir que o usuário calcule o consumo de aparelhos e estime o custo da energia com base no uso diário, mensal e anual.

---

## 🛠️ Ferramentas utilizadas

- Python
- VS Code
- GitHub

---

## ⚙️ Funcionalidades

- Cálculo de consumo diário, semanal, mensal e anual
- Estimativa de custo em reais (R$)
- Classificação do consumo:
  - Baixo consumo
  - Médio consumo
  - Alto consumo

---

## 📐 Fórmula utilizada

Consumo (kWh) = (Potência (W) × Horas por dia) / 1000

Custo = Consumo × valor do kWh

## ▶️ Como executar

### 🌐 Opção 1: Executar online (sem instalar nada)

1. Acesse: https://www.online-python.com/
2. Copie todo o conteúdo do arquivo `app.py`
3. Cole no editor do site
4. Clique em **Run**
5. Interaja com o programa pelo console exibido

---

### 💻 Opção 2: Executar com Python (terminal)

1. Baixe o arquivo `app.py` deste repositório
2. Abra a pasta onde o arquivo está salvo
3. Abra um terminal nessa pasta
4. Execute o comando:

```bash
python app.py
```

---

### 🧑‍💻 Opção 3: Executar com VS Code

1. Abra o VS Code  
2. Clique em **File > Open Folder**  
3. Selecione a pasta do projeto  
4. Abra o arquivo `app.py`  
5. Clique no botão **Run ▶️** no canto superior direito
